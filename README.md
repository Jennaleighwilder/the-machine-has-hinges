# The Machine Has Hinges

Standalone static website for a data center accountability field kit.

Open `index.html` directly in a browser, or serve the folder with:

```sh
python3 -m http.server 8080
```

The site is intentionally law-and-records focused:

- national map and all-state first-door finder
- cinematic stakes section for water, power, land, and subsidy impacts
- human-level living-world section with river video, public meeting, drought, and habitat imagery
- interactive watershed exhibit for peak heat, water stress, bill risk, and public-hearing leverage
- canvas graphs for energy and water teaching moments
- interactive impact lab for MW, water, subsidies, and permanent jobs
- interactive protection lens for residents, water systems, school budgets, and habitat
- seven-day field sprint checklist with progress
- public imagery/video section with NASA and Wikimedia assets
- public records request templates
- utility commission comment template
- meeting script
- moratorium language
- community benefit agreement demands
- procurement standards
- pressure-point and legal-thread maps
- Tennessee/Appalachia monitoring links as one regional example

Sources are linked in the page footer/source library. Keep factual claims dated and tied to those sources before publishing.
